# Portofolio-BE
# Langkah-langkah Instalasi

1. Buka terminal editor dan ketikkan

    ```bash
    git clone https://github.com/caawa/Portofolio-BE.git
    ```

2. Setelah itu masuk ke direktori project dan ketikkan

    ```bash
    npm install
    ```

3. Terakhir, jalankan project

    ```bash
    npm run start
    ```
